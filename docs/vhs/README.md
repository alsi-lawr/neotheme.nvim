# Reproducing the recordings

- [`neotheme-browser.tape`](neotheme-browser.tape) generates
  [`docs/assets/neotheme-browser.webp`](../assets/neotheme-browser.webp).
- [`neotheme-palette-editor.tape`](neotheme-palette-editor.tape) generates
  [`docs/assets/neotheme-palette-editor.webp`](../assets/neotheme-palette-editor.webp).

The palette-editor tape uses `.agent-workspace/vhs-palette-editor/` for isolated disposable state.

Use the WebP capture support from
[`alsi-lawr/vhs`](https://github.com/alsi-lawr/vhs/tree/feat/xterm6-capture-modes).
